Problem Statement

Gym Subcriptions can be really annoying as they are expensive and the trainers charge extra money for personal training, home workouts on the other hand prove to be both Cheat and time saving but can cause serious injuries if workout sessions are done with incorrect posture.

So to solve this problem we created Sensei which is a Computer Vision based Webapp which actively tracks User's Posture to provide feedback.
